# Verkada Access Control MCP (PM Hub)

Interactive overview of the custom Access Control MCP server built by **Jake Leichtling** (`jake.leichtling@verkada.com`).

Live hub: https://ankush-rustagi.github.io/verkada-ac-mcp/

Source code: https://github.com/verkada/docs-vibes/tree/main/35-custom-ac-mcp

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173/verkada-ac-mcp/

## Deploy

Push to `main` on the `verkada-ac-mcp` GitHub repo with GitHub Pages enabled (GitHub Actions workflow included). The site serves at `/verkada-ac-mcp/` on `ankush-rustagi.github.io`.
